# 极懒计划表

一个纯本地存储、无账号、无云端、极简、零多余功能的"每日计划表"网页工具。

## 项目简介

极懒计划表是一个专注于简洁和隐私的待办事项管理工具。所有数据仅存储在本地浏览器中，不会上传到任何服务器。这是一个单页面应用，无需后端支持，只需静态服务器即可部署。

## 功能特性

### 核心功能

- **日历导航**：点击日历中的日期快速切换查看不同日期的计划
- **任务管理**：
  - 添加任务：在输入框中输入任务内容，按 Enter 键添加
  - 完成任务：点击任务前的复选框，任务会移动到已完成区域
  - 编辑任务：双击任务文本即可编辑
  - 删除任务：点击任务右侧的删除按钮（×）
- **已完成任务**：默认折叠显示，可展开查看已完成的任务列表

### 视觉标记

日历中的日期会根据任务状态显示不同颜色的标记点：
- **蓝色点**：有计划（有未完成任务）
- **红色点**：过期未完成
- **绿色点**：过期已完成（仅已完成任务，无未完成任务）
- **蓝色边框**：今天

### 完成任务特效

完成任务时支持两种特效（可在设置中开关）：
- **动画效果**：任务缩小并淡出（默认开启）
- **音效**：播放完成音效（默认关闭）

### 设置选项

通过右上角的设置按钮可以配置：

- **数据管理**：
  - 备份：导出所有数据为 JSON 文件
  - 导入：从 JSON 文件恢复数据
- **顶部提醒**：控制是否显示本地存储警告提示
- **完成任务特效**：
  - 动画开关
  - 音效开关
- **外观**：Light / Dark 主题切换
- **支持开发者**：展示赞赏二维码（如果存在）

## 技术实现

### 技术栈

- **纯前端**：HTML + CSS + JavaScript（原生，无框架）
- **数据存储**：localStorage
- **无依赖**：不依赖任何外部库或框架

## 使用方法

### 本地运行

1. 克隆或下载项目到本地
2. 使用任意静态服务器打开 `index.html`，例如：
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - 或直接在浏览器中打开 `index.html`（某些功能可能受限）

### 部署

将项目文件上传到任意静态网站托管服务即可，例如：
- GitHub Pages
- Netlify
- Vercel
- 或任何支持静态文件的服务器

## 数据安全

⚠️ **重要提示**：

- 所有数据仅存储在本地浏览器的 localStorage 中
- 清理浏览器数据或更换设备将导致数据永久丢失
- 建议定期使用"备份"功能导出数据并妥善保存
- 数据不会同步或上传到云端

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge 等），需要支持：
- localStorage API
- ES6+ JavaScript 特性

## 注意事项

- 首次打开页面时会显示数据存储警告提示
- 每次重新打开页面都会默认显示今天的计划
- 已完成的任务默认折叠，需要点击 "Completed" 展开查看
- 导入数据时会验证数据格式，格式不正确会保持原数据不变
